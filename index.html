Flexbox can only be used to create one-dimensional layouts and CSS Grid can be used to create two-dimensional layouts. That means Flexbox can only create a layout vertically or horizontally, but not both at the same time.

Other side grid can create layout at the same time both direction vertically and horizontally

TailwindCSS is a utility-first CSS framework that packed css rule to utility class.
also TailwindCSS provides Bootstrap like components library but you have to pay for this.
Other hand Bootstrap offers pre-built interface components library like navigation, typography, buttons, forms and much more.

In CSS, the term "box model" is used when talking about design and layout. The CSS box model is essentially a box that wraps around every HTML element.
It consists of margins, borders, padding, and the actual content.

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible and meaningful by better defining the different sections and layout of web pages.

such as for header section semantics provide header tag, also article, aside, details, figcaption, figure, footer, header, main, mark, nav, section, summary, and time.
